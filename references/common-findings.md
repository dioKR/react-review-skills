# Common Findings Registry

Track recurring review findings here. When a pattern appears 3+ times, consider converting it to a lint rule or convention doc entry.

## How To Use

- Each entry has an ID, description, and occurrence count.
- Reference the ID in review comments (e.g., "See CF-001").
- After each review, update counts or add new entries.
- When count reaches 3+, mark as `→ lint candidate` and suggest automation.
